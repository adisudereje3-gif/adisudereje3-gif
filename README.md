<h2> 1. ATM Simulation System (Python)</h2>
An interactive console-based application that mimics the functionality of a real-world Automated Teller Machine.
Authentication: Secure PIN-based login system.
Transactions: Supports balance inquiry, cash deposits, and withdrawals.
Logic: Implements robust error handling to prevent over-drawing and invalid inputs.

<h2>2. Student Grade Calculator (Python & SQL)</h2>
A comprehensive tool for academic performance tracking.
Automated Grading: Converts numerical scores into academic grades ( A+, A, A-, B, C, etc.) using precise logical scales.
Performance Analysis: Calculates GPA and categorizes student standing (Excellent, Good, Poor).
Dual Implementation: Features both a Python script for quick logic and a SQL Scalar Function for database integration.

<h2>3. Department Placement Database (SQL Server)</h2>
A relational database schema designed for university department placements.
Relational Mapping: Tables for Students, Departments, Companies, and Placements.
Business Logic: Includes Stored Procedures for adding students and Triggers to prevent duplicate placements or ineligible entries (based on CGPA).
